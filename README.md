## Tech Stack
- Built with Kotlin / Compose / ViewModel  
- Dependency Injection (Koin)  
- Local Storage (Room)  
- Network Communication (Ktor)  

## Features
- Browse movie list  
- Search and sort  
- Pagination  
- Bookmark
- Support for both portrait and landscape modes
